# AnwendungGetraenkmarkt

Java-Anwendung zur Verwaltung eines Getränkemarktes mit grafischer Oberfläche (JavaFX).

## Funktionen
- Anlegen und Anzeigen von Getränken
- Import von Getränkedaten aus CSV- und TXT-Dateien
- Export der Daten in eine CSV-Datei
- Zwei GUI-Fenster (Eingabe & Warenübersicht)

## Architektur & Konzepte
- MVC-Architektur
- Design Patterns: Singleton, Observer, Factory Method
- Datei-Ein- und -Ausgabe (CSV, TXT)
- JavaFX GUI
- Unit-Tests mit JUnit

## Hinweis
Dieses Projekt entstand im Rahmen des Informatikstudiums und dient der praktischen Anwendung
von Software-Engineering- und Java-Konzepten.
